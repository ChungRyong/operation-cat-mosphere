## Tips for Nano Banana 2

1. **Style Anchor:** Generate **Cheese Cat** first to establish the art style. Use it as a reference image for all subsequent generations
2. **Sharpness:** If output has blurry edges despite keywords, try adding `"retro 16-bit SNES style"` or `"GameBoy Advance sprite"` to force cleaner pixel rendering
3. **Iteration:** Generate 4+ variations per character, pick the best, then use img2img or inpainting to refine
4. **Post-processing:** AI output will need manual cleanup — remove artifacts, fix pixel alignment, ensure transparent background with magic wand tool
5. **Sprite Sheet:** AI cannot reliably generate multi-frame sprite sheets — generate individual poses and assemble manually
6. **Color Palette:** Extract Cheese Cat's palette after finalizing, reference those exact hex codes in subsequent prompts for visual consistency
7. **Cat Identity:** Every man-made structure in this world was built by cats — always include cat furniture / cat toy elements in towers, buildings, and UI
8. **Tower Floor Height:** Each tower floor is 96x64 px — tall enough for decorative detail (cat beds, scratching posts, weapon mounts)
