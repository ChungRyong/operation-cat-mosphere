---
description: Wrap up current work — commit changes and compact conversation
---

Do the following steps in order:

1. **Check changes**: Run `git status` and `git diff` to see all modifications.

2. **Commit**: If there are uncommitted changes:
   - Analyze the diff to understand what was done
   - Write a concise commit message (Korean context OK, but commit message in English)
   - Stage relevant files and commit
   - If no changes exist, skip this step

3. **Compact**: After committing, run `/compact 잔여 작업 위주로 요약해줘` to compress the conversation with focus on remaining tasks.
